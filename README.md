## ğŸš€ Installation & Setup

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- (Optional) [Node.js](https://nodejs.org/) for development server

### Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/brianwalkerdev/responsive-web-page-html-css.git
   cd responsive-web-page-html-css
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or use a local development server (recommended):
   ```bash
   npm start
   ```

3. **View the site**
   - Direct: Open `index.html` in your browser
   - Live Server: Automatically opens at `http://localhost:8080`
